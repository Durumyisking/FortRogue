## Upgrades And Skills

ShellShock Live의 성장 요소는 별도 캐릭터 퍽보다는 탱크 업그레이드 트리로 구성된다. 레벨업할 때 업그레이드 토큰을 받아 `Chassis`, `Track`, `Gun`, `Luck`에 투자하고, 7일마다 토큰 재분배가 가능하다고 설명되어 있다.

| Tree | Main effect | Skills |
| --- | --- | --- |
| Chassis | 시작 HP/방어력 쪽 생존력을 올린다. 포인트당 시작 HP가 방어력 형태로 1% 증가하고, 최대 투자 시 시작 HP가 40% 추가된다. | Heavy Plating: 턴마다 낮은 확률로 방어력 회복. Advanced Internals: 턴마다 낮은 확률로 HP 회복. Adaptive Electronics: 턴마다 낮은 확률로 HP와 방어력 동시 회복. |
| Track | 경사 지형 이동 능력을 올리고, 스킬로 이동 연료를 늘린다. | Expanded Fuel Tank: 추가 연료. Fuel Efficient Engine: 효율 증가로 실질 이동량 증가. Pressurized Fuel Tank: 더 큰 추가 연료. |
| Gun | 무기 치명타 확률을 올린다. 최대 투자 시 치명타 확률은 20%로 설명되어 있다. | Reinforced Barrel: 치명타 배율 증가. Anti-Tank Missiles: 치명타 배율 추가 증가. Helical Grooves: 치명타 배율 추가 증가. 세 스킬을 모두 얻으면 치명타가 일반 피해의 2배까지 오른다고 정리되어 있다. |
| Luck | 매치 시작/상자에서 더 높은 단계 무기와 더 좋은 상자 보상을 얻을 확률을 올린다. | Hefty Haul: 상자 방어력 보상 가치와 상한 증가. Extra Items: 일회용 아이템 드롭 확률 증가. Grappling Hook: 가까운 상자를 자동 획득할 확률 추가. |
